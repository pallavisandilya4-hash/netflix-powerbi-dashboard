# Netflix Content Analysis Dashboard🎬

## Project Overview

This project is an interactive **Netflix Content Analysis Dashboard** built using **Power BI** to analyze trends, patterns, and insights from Netflix’s content library through dynamic and interactive visualizations.

The dashboard focuses on:

* Content growth trends over time
* Genre distribution and popularity
* Ratings analysis
* Movie vs TV Show comparison
* Country-wise content production
* Audience targeting patterns

The dashboard was designed using a **Netflix-inspired dark UI theme** to create a modern and engaging user experience.

---

# Tools & Technologies Used

* **Power BI** – Dashboard creation & visualization
* **Power Query** – Data cleaning & transformation
* **Data Visualization & Storytelling**
* **Interactive Dashboard Design**

---

# Dataset Information

The dataset contains information about Netflix titles including:

* Title Name
* Type (Movie / TV Show)
* Genre
* Rating
* Country
* Release Year
* Date Added
* Duration
* Cast & Director Information

#  Dashboard Design Approach

The dashboard was designed to combine analytical storytelling with interactive visualizations. Each page focuses on a specific analytical perspective, allowing users to explore Netflix content trends in a structured and intuitive manner.

Special attention was given to:

* Consistent Netflix-inspired UI styling
* Interactive slicers and filtering
* Visual hierarchy and readability
* Insight-focused storytelling
* Cross-page analytical flow

* Responsive visual storytelling

# Key Insights

* Movies dominate Netflix’s content library.
* Netflix content additions increased significantly after 2015.
* TV-MA and TV-14 are among the most common ratings.
* The United States contributes the highest amount of content.
* Most TV Shows contain fewer seasons compared to long-running series.
